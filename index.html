Installation

Using CDN

Include the following link in your HTML file to use AweSomeSlider without downloading

            
                <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/sikardp/AweSomeKitSlider/dist/AweSomeKitSlider.min.css">
                <script src="https://cdn.jsdelivr.net/gh/sikardp/AweSomeKitSlider/dist/AwesomeKitSlider.min.js"></script>

Using NPM

Install via npm for easy integration into your project

            
                npm i awesomekitslider

Then, import it into your JavaScript file

            
                import 'awesomekitslider/dist/AweSomeKitSlider.min.css';
                import AweSomeSlider from 'awesomekitslider';

Usage

1. HTML Markup

Start by adding the basic structure for your slider in your HTML file

            
                <div class="awe-slider-container" id="Basic">
                    <button class="awe-btn awe-prev">❮</button>
                    <div class="awe-slider">
                        <div class="awe-slide" style="background: #FF6F61;">Slide 1</div>
                        <div class="awe-slide" style="background: #6B5B95;">Slide 2</div>
                        <div class="awe-slide" style="background: #88B04B;">Slide 3</div>
                        <div class="awe-slide" style="background: #F7CAC9;">Slide 4</div>
                        <div class="awe-slide" style="background: #92A8D1;">Slide 5</div>
                    </div>
                    <button class="awe-btn awe-next">❯</button>
                </div>

2. Initializing the Slider

Once the HTML structure is in place, initialize the slider using JavaScript

            
                new AweSomeSlider('#Basic', {
                    arrows: true,
                });

3. Full Example

Here’s the complete implementation, including styles and script

                <html lang="en">

Properties

AweSomeSlider Properties

Below are the available properties in AweSomeSlider, along with their descriptions and usage

autoplay

Type : Boolean

Default : False

Description : Enables automatic sliding.

interval

Type : Number

Default : 3000 ( milliseconds )

Description : Sets the autoplay speed.

infinite

Type : Boolean

Default : False

Description : Enables infinite looping of slides.

arrows

Type : Boolean

Default : False

Description : Enables previous and next navigation arrows.

pagination

Type : Boolean

Default : False

Description : Enables pagination dots below the slider.

paginationText

Type : Array

Default : []

Description : Custom text for pagination items.

type

Type : string

Default : slide

Options : slide, fade

Description : Defines the animation type.

slidesPerView

Type : Number

Default : 1

Description : Sets the number of slides visible at a time.

slidesToScroll

Type : Number

Default : 1

Description : Sets how many slides move per transition.

breakpoints

Type : Object

Default : {}

Description : Allows different settings for different screen sizes.
